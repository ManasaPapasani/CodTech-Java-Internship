# CodTech Java Internship

This repository contains all four internship tasks completed using Java.

## Task 1 – File Handling Utility
Java program to read, write and modify text files.

## Task 2 – REST API Client
Java application that consumes a public REST API and parses JSON data.

## Task 3 – Multithreaded Chat Application
Client-server chat application using Java Sockets.

## Task 4 – AI-Based Recommendation System
Simple recommendation engine built using Java collections and logic.

All tasks are completed as per internship guidelines.
